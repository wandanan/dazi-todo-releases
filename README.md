## 产品定位

**数字秘书**是数字员工的基础形态，一个常驻桌面的 AI 工作助手。

它由 **Skill 智能体** + **Rust 桌面应用** 组成，7×24小时在本地运行，接管你的工作事务管理：

- 🎯 **自动规划**：根据你的工作风格智能安排日程
- 📊 **全程跟进**：自动跟踪进度、识别阻塞、推送提醒
- 🎨 **个性表达**：节气主题、心情卡片，让工作有温度
- 📄 **一键汇报**：日报、周报、月报、总结自动生成

**核心设计理念**：把工作丢给它，它会自己搞定一切。

## 核心功能

### 1️⃣ 智能日程规划（自动安排）

无需手动整理，直接把工作丢给秘书：

```
你： "下周要交设计稿，还有三个会议要准备"

秘书：
  ✓ 分析任务优先级和依赖关系
  ✓ 根据你历史工作风格分配时间段
  ✓ 预留缓冲时间应对突发情况
  ✓ 在面板生成今日执行清单
```

**学习你的工作风格**：
- 晨型人 vs 夜猫子（自动安排重要任务时段）
- 深度工作偏好（自动预留连续时间段）
- 会议疲劳阈值（自动限制每日会议数）

### 2️⃣ 全程进度跟进（主动服务）

不是被动等待，而是主动推进：

| 场景 | 秘书行为 |
|------|----------|
| 每天开机 | 自动更新今日工作，推送执行清单 |
| 任务到期前 | 提前提醒，询问是否需要延期或协助 |
| 检测到阻塞 | 主动询问障碍，协调资源或调整计划 |
| 长期挂起 | 风险提示，询问是否取消或重新安排 |
| 工作间隙 | 智能推荐下一步行动（基于优先级和精力） |

### 3️⃣ 风格化卡片（情感连接）

让工作助手有温度，不只是工具：

**节气主题卡片**
- 立春、清明、夏至、中秋... 24节气自动切换主题
- 结合节气特点给出工作建议（"春分宜平衡，今日轻重任务搭配"）

**心情定制卡片**
- 今日心情：🌟高能 / 😴疲惫 / 😤烦躁 / 🧘平静
- 根据心情调整任务推荐和界面配色
- 疲惫日自动减少任务量，推荐轻松事项

**增值服务（未来）**
- 节日限定主题包
- 企业品牌定制
- 个人照片/插画定制

### 4️⃣ 智能报告生成（一键总结）

告别月底写总结的痛苦：

```bash
# 自动生成日报
"生成今日日报"
→ 输出：今日完成任务、明日计划、阻塞问题

# 自动生成周报
"生成本周周报"
→ 输出：本周成果、进度概览、下周重点

# 自动生成月报
"生成本月总结"
→ 输出：月度成果、时间分布图、效率分析
```

**报告内容自动包含**：
- 完成的任务清单（带时间戳）
- 工作时长统计
- 会议参与情况
- 项目进度更新
- 待办事项清理率

## 典型工作流

### 早晨开机场景

```
08:30 你打开电脑

数字秘书：
  1. 检测 todo.md 变更，同步最新状态
  2. 分析今日待办，结合你的日程和风格
  3. 弹出面板："早上好！今日3项重点任务，建议从设计评审开始"
  4. 托盘静默，随时待命
```

### 工作进行中

```
10:00 你正在写代码

数字秘书（检测到专注状态）：
  → 自动开启勿扰模式
  → 延迟非紧急通知
  → 25分钟后弹出："专注时间结束，休息5分钟？"

11:30 会议前10分钟

数字秘书：
  → 弹出通知："会议即将开始：产品评审会"
  → 面板自动显示会议相关准备材料
```

### 工作交接场景

```
17:30 下班前

你："今天工作结束"

数字秘书：
  1. 生成本日工作摘要
  2. 标记已完成任务 ✅
  3. 将未完成项自动迁移到明日计划
  4. 询问："有阻塞需要明天优先处理的事项吗？"
  5. 更新 todo.md，同步到云（如配置）
```

### 月底总结场景

```
月底最后一天

你："生成本月工作总结"

数字秘书：
  1. 扫描整月 todo.md 历史
  2. 统计：完成任务数、项目里程碑、会议参与
  3. 识别：高频工作类型、效率趋势、阻塞模式
  4. 生成：带图表的月度报告
  5. 输出：可直接提交上级的格式（Markdown/Word/PDF）
```

## 技术实现

### Agent 常驻方案

**方案 A：Claude Code / OpenClaw 模式**
```bash
# 启动长驻 Agent 进程
claude-code --daemon --skill=./skills/todo_skill.yaml

# Agent 持续运行：
# - 监听文件系统事件
# - 定时检查日程
# - 触发面板行为
```

**方案 B：本地轻量 Agent**
```bash
# Python/Rust 实现的本地 Agent
python -m dazi.agent.daemon --mode=secretary

# 特点：
# - 资源占用低（<100MB）
# - 完全离线运行
# - 支持自定义模型（Ollama/本地LLM）
```

### 双进程通信

```
┌─────────────┐      HTTP/1431      ┌─────────────┐
│   Agent     │ ◄────────────────► │   面板 APP   │
│  (后台常驻)  │                    │  (桌面可视)  │
└─────────────┘                    └─────────────┘
      │                                  │
      └────────── 共享 todo.md ──────────┘
```

## 项目模块

数字秘书由两大核心模块组成：

### 🤖 AI Agent 智能体
负责理解用户需求、规划任务、主动服务
- **日程规划器** - 智能安排任务时间和优先级
- **进度跟踪器** - 监控任务状态，识别风险
- **报告生成器** - 自动生成日报/周报/月报
- **风格学习器** - 学习用户工作习惯，个性化推荐

### 🖥️ 桌面面板
负责可视化呈现、系统级交互
- **浮窗界面** - 轻量待办展示，支持拖拽定位
- **系统托盘** - 常驻后台，快速唤起
- **通知中心** - 日程提醒、任务预警
- **专注模式** - 番茄钟计时，沉浸式工作
- **主题卡片** - 节气主题、心情配色

### 📦 数据工作区 (Workspace)
所有用户工作数据统一存放在 `workspace/` 目录：
- **todo.md** - 当前待办事项，Agent 每日维护
- **work_memory.csv** - 工作记忆，长期归档已完成任务
- **soul.md** - 用户偏好与默契档案

### 🎒 技能包 (Skills)
- 数字秘书 Skill - Agent 行为定义和场景配置
- 可扩展其他专业 Skills（项目管理、写作助手等）

| 文件 | 说明 |
|------|------|
| `app/dazi-todo.exe` | 主应用（GUI 浮窗） |
| `app/dazi-todo-cli.exe` | 命令行控制工具 |
| `SKILL.md` | Agent 集成文档 |
| `workspace/` | 工作区（用户数据） |

## CLI 命令示例

```bash
# 显示/隐藏窗口
./app/dazi-todo-cli.exe show
./app/dazi-todo-cli.exe hide

# 添加待办
./app/dazi-todo-cli.exe add "任务内容"

# 列出待办
./app/dazi-todo-cli.exe list

# 调整窗口大小
./app/dazi-todo-cli.exe size 400 600

# 设置窗口位置
./app/dazi-todo-cli.exe position bottom-right

# 启动番茄钟（专注模式）
./app/dazi-todo-cli.exe focus "写代码" 25

# 设置自动关闭时间（秒）
./app/dazi-todo-cli.exe autoclose 60
./app/dazi-todo-cli.exe autoclose 0  # 禁用自动关闭
```
